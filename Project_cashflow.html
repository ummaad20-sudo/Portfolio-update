06 · Fitur Aplikasi
MODUL & FITUR UTAMA APLIKASI BASE44 PLATFORM
📝
Pencatatan Transaksi
Input pemasukan & pengeluaran dengan kategori, nama pihak, uraian, dan nominal. Mendukung multi-kategori: Operasional, Gaji, Project, Iuran, Lain-lain.
📊
Visualisasi Keuangan
Grafik bar interaktif cash flow per bulan, tampilan tren keuangan tahunan, dan breakdown per kategori untuk analisa mendalam.
🗂
Filter & Pencarian
Filter transaksi berdasarkan kategori, bulan, atau nama pihak. Navigasi cepat antar bulan dengan selector bulan interaktif.
📤
Import & Ekspor CSV
Import data transaksi massal via file CSV dan ekspor laporan keuangan lengkap untuk kebutuhan audit atau pelaporan manajemen.
📱
Real-time Monitoring
Saldo dan summary terupdate otomatis setiap ada transaksi baru. Dashboard dapat diakses dari browser desktop maupun mobile.
🤖
AI Advisor
Fitur konsultasi AI terintegrasi untuk analisa kondisi keuangan, saran pengelolaan arus kas, dan rekomendasi berdasarkan data historis.
07 · Formula Kalkulasi
LOGIKA PERHITUNGAN SISTEM OTOMATIS
SALDO BULAN
Total Masuk
− Total Keluar
= Saldo Bulan
Dihitung otomatis per bulan dari seluruh transaksi yang tercatat dalam periode tersebut.
GRAND SALDO
Σ Semua Masuk
− Σ Semua Keluar
= Grand Saldo
Akumulasi seluruh transaksi sepanjang tahun — mencerminkan kondisi keuangan keseluruhan perusahaan.
DISTRIBUSI KATEGORI
Σ per Kategori
÷ Total Keluar
= % Distribusi
Persentase pengeluaran per kategori untuk mengetahui pos biaya terbesar dalam operasional perusahaan.

✅ KEUNGGULAN APLIKASI
  • Dikembangkan dengan Base44 — platform no-code/low-code yang memungkinkan development cepat tanpa infrastruktur server kompleks.
  • Saldo dan grafik terupdate real-time setiap ada input transaksi baru — tidak perlu refresh manual.
  • Fitur AI Advisor membantu manajemen mengambil keputusan keuangan berdasarkan data historis transaksi.
  • Sistem kategori fleksibel memudahkan pemisahan arus kas operasional, gaji, project, dan iuran perusahaan.
⚠ CATATAN PENGGUNAAN
  • Setiap transaksi wajib dilengkapi Nama Pihak, Uraian, Kategori, dan Nominal untuk konsistensi laporan.
  • Backup data secara berkala menggunakan fitur Ekspor CSV untuk mencegah kehilangan data.
  • Pastikan input nominal dalam satuan Rupiah (Rp) tanpa desimal untuk menghindari error kalkulasi.
  • Fitur Import CSV mengikuti format template yang disediakan — jangan ubah struktur kolom template.
