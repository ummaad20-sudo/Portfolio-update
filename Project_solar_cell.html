⚠ CATATAN PENTING INSTALASI
  • AC ½ PK bersifat induktif → wajib inverter Pure Sine Wave. Modified sine wave akan merusak AC dan pompa.
  • AC saat start (starting current) bisa 3–5× daya normal → pastikan inverter punya surge capacity ≥ 2.000W.
  • Jangan jalankan semua beban secara bersamaan. Total simultan maksimum ≤ kapasitas inverter.
  • Konfigurasi baterai: 2 unit 12V disambung SERI (+ ke − baterai 1 ke baterai 2) untuk mendapat 24V.
  • Konfigurasi panel: 2 panel seri (→ 44V) lalu 2 string paralel → input SCC ±44V, arus ±20A per string.
  • Pastikan instalasi dilakukan oleh teknisi berpengalaman. Sistem DC tegangan tinggi berbahaya.
  • Letakkan baterai di tempat berventilasi baik, jauh dari sumber api.
