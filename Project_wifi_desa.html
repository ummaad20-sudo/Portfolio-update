05 · Tahapan Konfigurasi Remote
REMOTE CONFIGURATION & DEPLOYMENT WORKFLOW STEP BY STEP
1
PERSIAPAN & SURVEY LOKASI
Survey kondisi lapangan: cek sinyal Starlink (obstruction map), identifikasi titik optimal pemasangan dish, ukur jarak antar titik AP, dan rencanakan jalur kabel. Pastikan tersedia sumber listrik di setiap lokasi desa.
2
AKTIVASI STARLINK
Pasang dish Starlink pada posisi tanpa hambatan langit (obstruction < 2%). Aktifkan melalui aplikasi Starlink App, lakukan registrasi akun, dan verifikasi koneksi satelit. Sambungkan output Starlink ke WAN port Router Ruijie menggunakan kabel Ethernet.
3
KONFIGURASI ROUTER RUIJIE — CLOUD ONBOARDING
Power on Router Ruijie → buka app.ruijienetworks.com atau Reyee App. Tambahkan perangkat menggunakan SN (Serial Number) router. Atur SSID WiFi, password, DHCP range, dan bandwidth per user dari dashboard cloud. Semua konfigurasi push secara remote tanpa perlu akses fisik ke perangkat.
4
PEMASANGAN & ADOPSI ACCESS POINT
Pasang Access Point Ruijie di titik optimal (ketinggian 3–5m). Sambungkan via kabel UTP Cat6 ke PoE Switch / Injector. AP akan otomatis terdaftar ke Router Ruijie dan muncul di dashboard Reyee Cloud. Lakukan adoptasi (Adopt) dan AP langsung aktif dengan konfigurasi yang sudah diset di router.
5
PENGUJIAN & VERIFIKASI JARINGAN
Uji koneksi dari perangkat user: speed test via fast.com atau speedtest.net, ping test ke server eksternal, dan verifikasi coverage WiFi di seluruh area desa. Pantau jumlah client terhubung dan kualitas sinyal dari dashboard Reyee Cloud secara real-time.
6
MONITORING & REMOTE MANAGEMENT BERKELANJUTAN
Pantau status semua perangkat di 3 desa melalui Reyee Cloud Dashboard secara berkala. Notifikasi otomatis dikirim jika ada perangkat offline. Update firmware perangkat dilakukan remote. Troubleshooting dan perubahan konfigurasi dapat dilakukan kapan saja tanpa perlu ke lokasi.
✅ KEUNGGULAN SOLUSI INI
  • Starlink menjangkau area terpencil yang tidak dapat dilayani ISP konvensional (fiber/kabel).
  • Ruijie Reyee memiliki ekosistem cloud management terintegrasi — 1 dashboard untuk semua lokasi.
  • Konfigurasi 100% remote menghemat biaya perjalanan berulang ke desa-desa terpencil.
  • Access Point Ruijie mendukung roaming seamless antar AP dalam satu jaringan yang sama.
  • Bandwidth control per user mencegah monopoli bandwidth oleh satu pengguna.
⚠ CATATAN PENTING INSTALASI
  • Dish Starlink wajib dipasang di tempat tanpa halangan langit — pohon, atap, atau bangunan tinggi mengurangi kualitas sinyal drastis.
  • Gunakan kabel UTP outdoor/weatherproof untuk instalasi luar ruangan agar tidak rusak akibat cuaca dan UV.
  • Pasang surge protector / lightning arrester pada jalur LAN dan listrik untuk melindungi perangkat dari sambaran petir.
  • Pastikan sistem grounding terpasang dengan benar pada setiap lokasi — daerah terpencil rentan petir.
  • Simpan catatan SN (Serial Number) setiap perangkat untuk memudahkan re-adoption jika ada penggantian perangkat.
  • Dokumentasikan password WiFi dan akun Reyee Cloud dengan baik dan simpan di tempat aman.
