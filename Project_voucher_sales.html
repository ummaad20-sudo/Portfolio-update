06 · Arsitektur Kode
STRUKTUR CLASS & METHOD UTAMA PYTHON · KIVY
CLASS: ColoredBox
Custom widget turunan BoxLayout dengan latar belakang berwarna. Menggunakan RoundedRectangle di canvas untuk sudut melengkung. Bind pos + size agar warna ikut saat resize.
CLASS: RekapApp — build()
Membangun seluruh UI secara programatik (tanpa .kv file). Menyusun 5 seksi: Header → Input Manual → Pencarian/Hapus → Hasil → Tombol Aksi.
METHOD: proses_excel()
Buka file .xlsx dengan openpyxl. Auto-detect header kolom menggunakan list.index(). Handle tanggal datetime & string. Append ke all_data list.
METHOD: update_display()
Core rekap engine. Gunakan defaultdict untuk grupkan data. Hasilkan dua string: markup Kivy + plain text. Hitung Grand Total akumulasi.
METHOD: hapus_data_tanggal()
Filter all_data menggunakan list comprehension — buang semua item dengan tanggal yang cocok. Tampilkan notifikasi jumlah data terhapus via show_notif().
METHOD: copy_ke_clipboard()
Salin hasil_text_bersih (plain text tanpa markup Kivy) ke clipboard menggunakan Clipboard.copy(). Auto-notif 3 detik.
07 · Tahapan Pengembangan
ALUR PENGEMBANGAN APLIKASI DEV WORKFLOW
1
ANALISA KEBUTUHAN
Identifikasi kebutuhan pengguna: rekap penjualan voucher WiFi harian, input cepat via HP Android, dan kemudahan kirim laporan via WhatsApp. Tentukan format data Excel dari sistem voucher yang digunakan.
2
PERANCANGAN UI (PYTHON KIVY)
Desain antarmuka menggunakan BoxLayout, ScrollView, dan TextInput secara programatik. Tema warna: biru muda (input), hijau muda (pencarian), putih (hasil). Custom widget ColoredBox dengan RoundedRectangle.
3
PENGEMBANGAN MODUL EXCEL
Integrasi openpyxl untuk membaca file .xlsx. Auto-detect kolom header menggunakan list.index(). Handle dua format tanggal: datetime object dan string. File chooser dengan filter *.xlsx.
4
ENGINE REKAP & KALKULASI
Implementasi defaultdict untuk pengelompokan otomatis per (tanggal, grup). Kalkulasi jumlah pcs, total rupiah, total per hari, dan Grand Total. Dua format output: markup Kivy untuk tampilan dan plain text untuk clipboard.
5
FITUR HAPUS & PENCARIAN
Tambahkan fitur hapus data per tanggal menggunakan list comprehension. Implementasi filter pencarian real-time. Notifikasi auto-dismiss 3 detik menggunakan Clock.schedule_once().
6
BUILD & DEPLOY ANDROID
Build APK menggunakan Buildozer atau python-for-android. Tambahkan permission Android: READ_EXTERNAL_STORAGE. Tes pada perangkat Android nyata, optimasi ukuran font dan layout untuk layar kecil.

✅ KEUNGGULAN APLIKASI
  • Cross-platform: satu kode Python berjalan di Android dan Desktop (Windows/Linux/macOS) tanpa modifikasi.
  • Auto-detect kolom Excel — tidak perlu atur urutan kolom, cukup nama header yang sesuai.
  • Dua format output (markup Kivy + plain text) memastikan hasil siap ditampilkan dan siap dikirim.
  • Fitur hapus per tanggal memudahkan koreksi data tanpa perlu hapus seluruh dataset.
  • Notifikasi auto-dismiss 3 detik memberikan feedback instan tanpa mengganggu alur kerja.
⚠ PERSYARATAN & CATATAN TEKNIS
  • Pastikan nama kolom di file Excel persis sama dengan yang didukung — "Grup pengguna", "Harga", "Diaktifkan di".
  • Untuk build APK Android, gunakan Buildozer dengan sistem operasi Linux (atau WSL di Windows).
  • Tambahkan android.permissions = READ_EXTERNAL_STORAGE di buildozer.spec untuk akses file di Android.
  • Kivy tidak mendukung semua widget native Android — gunakan KivyMD jika ingin tampilan Material Design.
